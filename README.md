# FileForge Pro

A polished file management console — create, read, update, and delete text files — built with Streamlit.

## Features
- **Create** — new file with initial content
- **Read** — search/filter files, preview text content inline, or download any file
- **Update** — rename, append, overwrite, or duplicate a file
- **Delete** — single-file delete or bulk delete with confirmation
- **Manage** — browse the workspace with search and sort (name / size / modified)
- **Activity log** — session log of recent actions
- **Stats strip** — live file count, total size, and last activity time

## Tech
Python · Streamlit

## Run locally
```bash
pip install streamlit
streamlit run app.py
```

Files are stored in a local `fileforge_files/` folder created automatically on first run.
